TweetGazer by とろゝ Readme

■このソフトウェアについて
　Windows PC用のTwitterクライアントアプリケーションです。

■動作環境
　□必須環境
　　・Windows OSの動作しているPC
　　・.NET Framework 4.6
　□テスト環境1
　　・OS          : Windows 10 Pro 64bit
　　・CPU         : Intel(R) Core(TM) i7-6700K CPU@4.00GHz 4.01GHz
　　・物理メモリ  : 16.00GB
　□テスト環境2
　　・OS          : Windows 10 Pro 64bit
　　・CPU         : Intel(R) Core(TM) i7-3667U CPU@2.00GHz 2.00GHz
　　・物理メモリ  : 8.00GB
